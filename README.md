# koala-cli

A command-line interface tool to manage your [KoalaVim](https://github.com/KoalaVim/KoalaVim) installation and its dependencies.

## Installation

```bash
python -m pip install git+ssh://git@github.com/KoalaVim/koala-cli.git
```
